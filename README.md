【baiduTopZz3】： 
百度搜索风云榜资讯爬虫ahk脚本3（论坛地址：http://ahk8.com/thread-6066.html）

【功能】：
阅读实时热点,
关注人群风向,
排除过往资讯,
批量打开浏览。

【建议】：
可将此脚本加入每日开机运行，如无最新未读新闻不会提示打扰，运行完自动退出。
可修改##自定义修改变量##适应个人情况
可增加个人感兴趣的资讯排行榜，在此挑选http://top.baidu.com/boards?fr=topregion
排行榜增加过多后，会根据个人网速下载网页数据的时间变长

【已经添加监控的资讯有】：
今日热门搜索排行榜
七日热点排行榜
今日电影排行榜
今日世说新词排行榜
